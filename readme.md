# Quick Start Guide

This will work in conjuction with the coursecareers-bookmarks-API. Have separate terminal open with server running using `npm run start`

Use favs in the command line to see a list of options to bookmark and open your favorite websites!

This is NOT for end users. This is for developer testing purposes only.

Includes SDK and SDK tests.

To use:
`favs <command> `

Commands:

- ls : List all favorites
- open <favorite> : Open a saved favorite
- add <favorite> <url> : add a new favorite for some URL
- rm <favorite> : remove a saved favorite.
